# h1-zombieland-mod

Zombieland mod for H1 (MWR)

Currently supports these maps:

- Bog

# How

- Install [h1-mod](https://github.com/h1-mod/h1-mod)
- Download this mod and place in the Modern Warfare Remastered\h1\scripts folder
